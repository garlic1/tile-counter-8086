# Wall Tile Counter For The 8086 Processor

## Details

This is a program with the purpose of learning the Assembly language for old Intel processors.

## Demo
[https://youtu.be/KuVeiN49mZ0](https://youtu.be/KuVeiN49mZ0)

## How to Run

- Download and install [DosBox](https://www.dosbox.com/download.php?main=1)
- Download the MASM assembler (available in the repo)
- Install MASM in your DosBox virtual machine
- Run `masm paredes.asm`
- Run `link paredes.obj`
- Run `paredes.exe`

### Usage:
- Upon entering the program, type an entry file name, i.e. `parede1.par` and press Enter (sample files are provided)
- Press Enter to quit the wall view
- Choose another file to open or press Enter to quit
